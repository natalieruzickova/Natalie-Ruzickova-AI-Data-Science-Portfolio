# 🛒 Customer Segmentation Using RFM Analysis & K-Means Clustering

## Overview

This project focuses on customer segmentation for an e-commerce business using the **RFM (Recency, Frequency, Monetary) model** combined with **K-Means clustering**.

The objective was to identify groups of customers with similar purchasing behaviors and provide actionable insights to support personalized marketing strategies, customer retention, and CRM optimization.

## Business Problem

E-commerce companies collect large amounts of transactional data but often struggle to understand customer value and behavior.

This project addresses the question:

> How can customer purchasing patterns be analyzed to create meaningful customer segments and improve marketing decisions?

## Methodology

The workflow included:

- Data preparation and cleaning
- Calculation of RFM metrics:
  - **Recency** – how recently a customer purchased
  - **Frequency** – how often a customer purchased
  - **Monetary** – how much a customer spent
- Data normalization and scaling
- Determining optimal cluster numbers using:
  - Elbow method
  - Silhouette score
- Applying K-Means clustering
- Analyzing and interpreting customer segments

## Results

The model identified several customer groups, including:

- 🏆 Champions
- ⭐ Potential Loyalists
- 🌱 Promising Customers
- ❄️ Hibernating Customers

The segmentation provided insights for:

- Targeted marketing campaigns
- Customer retention strategies
- Personalized communication
- Improved customer relationship management

## Technologies

`Python` `R` `Pandas` `K-Means` `RFM Analysis` `Data Visualization`

# 🛍️ Customer Segmentation Analysis - Online Retail II Dataset

## Overview

This project presents an end-to-end customer segmentation workflow using the Online Retail II dataset.

The goal was to transform raw transactional data into meaningful customer insights using RFM analysis and clustering techniques.

## Data Processing

The workflow included:

- Loading transactional data
- Handling missing values
- Removing cancelled transactions
- Calculating transaction totals
- Creating customer-level metrics

## RFM Analysis

Customer behavior was measured using:

- **Recency**
- **Frequency**
- **Monetary Value**

These features were standardized before applying clustering algorithms.

## Clustering Approach

The project used:

- Elbow method for cluster selection
- Silhouette score evaluation
- K-Means clustering
- Cluster visualization and interpretation

## Results

The analysis revealed different customer behavior patterns and demonstrated how transactional data can support:

- Customer targeting
- Marketing optimization
- Business decision-making

## Technologies

`Python` `Pandas` `Scikit-learn` `K-Means` `Data Visualization`
