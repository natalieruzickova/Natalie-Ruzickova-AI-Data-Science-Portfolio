# Customer Segmentation Using RFM Analysis & K-Means Clustering

## Repository

**GitHub:** https://github.com/natalieruzickova/bachelors-thesis

## Overview

This project focuses on customer segmentation for an e-commerce business using the **RFM (Recency, Frequency, Monetary)** model combined with **K-Means clustering**.

The objective was to identify groups of customers with similar purchasing behaviors and provide actionable insights to support personalized marketing strategies, customer retention, and CRM optimization.

## Business Problem

E-commerce companies collect large amounts of transactional data but often struggle to understand customer value and behavior.

This project addresses the question:

> How can customer purchasing patterns be analyzed to create meaningful customer segments and improve marketing decisions?

## Methodology

- Data preparation and cleaning
- RFM feature engineering
- Feature scaling
- Elbow Method
- Silhouette Score
- K-Means clustering
- Cluster interpretation

## Results

Customer segments identified include:

-  Champions
-  Potential Loyalists
-  Promising
-  Hibernating

The project provides recommendations for personalized marketing, customer retention, and CRM optimization.

**Technologies**

`Python` `Pandas` `K-Means` `RFM Analysis` `Data Visualization`
