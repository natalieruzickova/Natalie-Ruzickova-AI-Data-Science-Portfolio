# Customer Segmentation Using RFM Analysis & K-Means Clustering

**Repository**

https://github.com/natalieruzickova/bachelors-thesis

## Overview

This bachelor's thesis investigates customer segmentation for an e-commerce company using the RFM (Recency, Frequency, Monetary) model combined with K-Means clustering. The objective was to identify meaningful customer groups based on purchasing behavior and provide recommendations for personalized marketing strategies and customer relationship management.

## Business Problem

Understanding customer purchasing behavior is essential for improving retention, increasing customer lifetime value, and optimizing marketing campaigns. This project explores how customer transaction data can be transformed into actionable customer segments.

## Methodology

The project includes:

- Data cleaning and preprocessing
- Calculation of Recency, Frequency, and Monetary metrics
- Feature scaling
- Elbow Method for cluster selection
- Silhouette Score evaluation
- K-Means clustering
- Customer segment interpretation

## Results

The analysis identified distinct customer groups including:

- Champions
- Potential Loyalists
- Promising
- Hibernating

These segments provide valuable insights for:

- Personalized marketing campaigns
- Customer retention strategies
- CRM optimization
- Business decision-making

## Technologies

- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- RFM Analysis
- K-Means Clustering
